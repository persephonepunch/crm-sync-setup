# Self-hosted fonts

Variable-font woff2 builds served same-origin by this GitHub Pages site so no
document request ever reaches a third-party font CDN (see
[WAYFAIR-TO-AI-COMMERCE](../WAYFAIR-TO-AI-COMMERCE.md), field-checklist flag #2,
and LG München I, 3 O 17493/20).

| File | Family | Axes |
| --- | --- | --- |
| `roboto-vf.woff2` | Roboto | wght 100–900 · wdth 75–100 |
| `roboto-mono-vf.woff2` | Roboto Mono | wght 100–700 |

Converted from the upstream variable TTFs with fontTools (flavor=woff2).
